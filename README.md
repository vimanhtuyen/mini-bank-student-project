# Mini Bank (Console Application)

Dự án này phù hợp cho học sinh lớp 8: code rõ ràng, dễ đọc, chia phần theo module.

## 1) Yêu cầu
- Python 3.9 trở lên (khuyến nghị)

## 2) Cách chạy
Mở Terminal/CMD tại thư mục dự án và chạy:

```bash
python main.py
```

## 3) Dữ liệu
- Dữ liệu được lưu tại: `data/bank_data.json`
- Khi bạn tắt chương trình và chạy lại, dữ liệu vẫn còn.

## 4) Chức năng
- Tạo tài khoản (PIN 4–6 chữ số)
- Đăng nhập
- Nạp tiền / Rút tiền / Chuyển khoản
- Xem số dư
- Xem lịch sử giao dịch
